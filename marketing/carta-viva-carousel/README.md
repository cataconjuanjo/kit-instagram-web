# Carrusel Instagram - Carta Viva

Versión B: enfoque oportunidad de venta para restaurantes.

## Concepto

Idea central: **Convierte tu carta de vinos en una herramienta de venta para sala.**

El carrusel no vende "una app" como pieza técnica. Vende una mejora concreta para el restaurante: menos fricción para el cliente, más criterio para sala y más oportunidades de consultoría en la carta.

## Slides

1. Portada: convertir la carta en herramienta de venta.
2. Carta pública: del QR al pedido.
3. Modo camarero: acceso privado para sala.
4. Gestión: actualizar sin rehacer PDFs.
5. Importación: partir de la carta real.
6. Hub: sustituir el link en bio disperso.
7. Radar consultoría: oportunidades ocultas.
8. Málaga: territorio y diferenciación.
9. Demo real: prueba con la carta del cliente.
10. CTA: escribir "Carta Viva".

## Segmentación sugerida

Para probar en Instagram Ads:

- Ubicación: Málaga capital y provincia.
- Intereses: restaurantes, hostelería, gastronomía, vino, sumillería, hoteles boutique.
- Cargo/actividad si Meta lo permite: propietarios de restaurantes, gerentes, hostelería, restauración.
- Creatividad: formato carrusel 4:5.
- CTA del anuncio: "Enviar mensaje" o "Más información".

## Copy corto para el anuncio

Tu carta de vinos puede ser mucho más que un PDF.

Con Carta Viva, el cliente encuentra mejor, la sala recomienda con más criterio y tú detectas oportunidades reales de venta: copas, vino local, maridajes, postres y referencias que merecen estar mejor explicadas.

Escríbeme "Carta Viva" y preparo una demo con tu carta.
